WANDER DIARIES — Interactive Portfolio (Offline Version)
=========================================================

WHAT THIS IS
An interactive Instagram-feed portfolio sample. Tap any post to open it.
Videos and reels play. Carousels swipe. Works fully offline — all photos
and videos are bundled inside this folder.

HOW TO OPEN
1. UNZIP this folder first (don't open files from inside the zip).
2. Open the file named  index.html  by double-clicking it.
   - On a computer: it opens in your default browser (Chrome/Edge/Safari).
   - On a phone: unzip with the Files app, then tap index.html.
     (If your phone won't open it, the easiest fix is to host it as a
      link — see "Sharing as a link" below.)

WHAT'S INSIDE
  index.html      → the portfolio (open this)
  images/         → all 18 photos (1.jpg … 18.jpg)
  videos/         → the 5 video/reel clips (4, 8, 11, 13, 17)

IMPORTANT
Keep index.html, the images folder, and the videos folder together in the
same folder. If you move index.html out on its own, the photos and videos
won't show.

SHARING AS A LINK (best for the job application)
This works great as a hosted link. Free option:
  1. Go to app.netlify.com/drop
  2. Drag this whole unzipped folder onto the page.
  3. You'll get a link you can paste into the application email.
A hosted link opens instantly on any phone and looks the most professional.
